# Single-Page CV Project

This project is part of [roadmap.sh](https://roadmap.sh/projects/single-page-cv).  
It demonstrates how to build a structured, semantic CV using only HTML.

## Features
- Semantic HTML structure (header, main, section, article, footer).
- SEO meta tags for search optimization.
- Open Graph (OG) tags for social sharing.
- Favicon linked in the head section.
- Single-page layout with sections for Education, Skills, and Career History.
